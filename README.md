# Citation Formatter

An interactive web tool that extracts bibliographic information from notes and reformats them according to various citation styles.

## Features

- **Smart Parsing**: Automatically extracts author, title, journal, volume, issue, year, and page information
- **Multiple Citation Styles**: 
  - Chicago Manual of Style
  - MLA Format
  - APA Style
  - Bluebook Legal
  - Turabian
  - Keep Original Style
- **Visual Feedback**: Shows extracted information for verification
- **Copy to Clipboard**: One-click copying of formatted citations
- **Clean Interface**: User-friendly design with example formats

## Usage

1. Paste your citation or note into the text area
2. Select your desired citation format from the dropdown menu
3. Click "Format Citation"
4. Review the extracted information and formatted output
5. Copy the formatted citation to your clipboard

## Example Input Formats

```
Eric Caplan, "Trains, Brains, and Sprains," Bulletin of the History of Medicine 69(3):387-419 (1995)
```

```
Smith, John. The History of Medicine. New York: Oxford University Press, 2020, pp. 45-67
```

```
Author: Jane Doe | Title: Modern Psychology | Journal: Psychology Today | Volume: 12 | Issue: 3 | Year: 2023 | Pages: 100-115
```
