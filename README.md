# Smart Citation Generator

A web-based tool that automatically searches for bibliographic information and generates properly formatted citations.

## Features

- **Automatic Search**: Enter author and partial title, get complete bibliographic data
- **Multiple Citation Formats**: Chicago, MLA, APA, Bluebook, Turabian
- **Manual Entry Option**: Fill in details yourself if needed
- **One-Click Copy**: Copy formatted citations to clipboard
- **Clean Interface**: Easy to use, mobile-friendly

## How to Use

1. Enter author and partial title (e.g., "Eric Caplan Mind")
2. Click "Search for Book"
3. Select the correct work from results
4. Choose your citation format
5. Copy the formatted citation
